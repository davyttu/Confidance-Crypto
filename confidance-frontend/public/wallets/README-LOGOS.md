# Logos officiels des wallets

Pour afficher les **vrais logos** Trust Wallet et Coinbase (au lieu des versions simplifiées), télécharge les SVG officiels et enregistre-les ici.

## Trust Wallet

**Source officielle :**
- Page Press : **https://trustwallet.com/press**
- Zip des logos : **https://trustwallet.com/download/logos.zip**  
  → Ouvre le zip, récupère le SVG qui te convient, renomme-le en **`trust-wallet.svg`**.

**Alternative (SVG direct) :**
- **https://www.svgrepo.com/download/505018/trust-wallet.svg**  
  → Clic droit → « Enregistrer le lien sous » → enregistre sous **`trust-wallet.svg`**.

Place le fichier dans : **`confidance-frontend/public/wallets/`** (remplace l’actuel si besoin).

## Coinbase Wallet

**Source officielle :**
- Page Press : **https://www.coinbase.com/press**  
  → Télécharge le kit de marque et récupère le SVG de l’icône Coinbase Wallet.

**Alternatives (SVG direct) :**
- Icône : **https://www.vectorlogo.zone/logos/coinbase/coinbase-icon.svg**
- Logo : **https://www.svgrepo.com/download/330202/coinbase.svg**  
  → Clic droit → « Enregistrer le lien sous » → enregistre sous **`coinbase.svg`**.

Place le fichier dans : **`confidance-frontend/public/wallets/`** (remplace l’actuel si besoin).

## Vérification

Après avoir remplacé les deux fichiers, recharge la page « Première étape » : Trust Wallet et Coinbase Wallet afficheront les logos que tu as téléchargés (les chemins sont déjà configurés dans le site).
