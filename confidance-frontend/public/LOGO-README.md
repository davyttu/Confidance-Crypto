**Couleurs du dégradé :** `#6366f1` → `#a855f7` → `#ec4899`

**Pour obtenir un PNG :**
- Ouvrir le SVG dans un navigateur ou un logiciel (Figma, Inkscape, etc.), puis exporter en PNG à la taille souhaitée (ex. 2× pour écrans retina : 440×96 px).
- Ou utiliser un convertisseur en ligne (ex. cloudconvert.com, svgtopng.com).
